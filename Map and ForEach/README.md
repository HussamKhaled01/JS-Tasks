# 🧠 JavaScript Practice — `map()` & `forEach()`

## 📌 Instructions

* Use **`.map()`** when:

  * You need to transform data and return a new array

* Use **`.forEach()`** when:

  * You are calculating a single value (sum, count, etc.)
  * You don’t need to return a new array directly
